# USER SERVICE

1. Before running the service, create and .env file and add the following values:
```
MOVIE_MATE_USER_SERVICE_PORT= # optional (default 7030)
MOVIE_MATE_USER_SERVICE_CONTEXT_PATH= # optional (default /api/v1)

# DATASOURCE
MOVIE_MATE_USER_SERVICE_DATASOURCE_URL=
MOVIE_MATE_USER_SERVICE_DATASOURCE_USERNAME=
MOVIE_MATE_USER_SERVICE_DATASOURCE_PASSWORD=

# REDIS
MOVIE_MATE_USER_SERVICE_REDIS_HOST= # optional (default 6379)
MOVIE_MATE_USER_SERVICE_REDIS_PORT=
MOVIE_MATE_USER_SERVICE_DATABASE_INDEX=  # optional (default 0)
MOVIE_MATE_USER_SERVICE_REDIS_PASSWORD=
MOVIE_MATE_USER_SERVICE_REDIS_TTL=

# JWT
MOVIE_MATE_JWT_SECRET_KEY=
MOVIE_MATE_JWT_ACCESS_EXPIRATION_TIME=
MOVIE_MATE_JWT_REFRESH_EXPIRATION_TIME=

# ROOT USER
MOVIE_MATE_USER_SERVICE_ROOT_USERNAME=
MOVIE_MATE_USER_SERVICE_ROOT_EMAIL=
MOVIE_MATE_USER_SERVICE_ROOT_PASSWORD=
```
